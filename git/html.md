#CSS
Start from adding a style attribute.

```
<h1 style="color:blue;text-align:center">Welcome to My Web Page</h1>
```
color attribute 可以用 hex code

除了這種包在裡面的之外，還可以用
```
<style>
    h1 {
    color:blue;
    text-align:center;
    
    }
</style>
```

這種寫法會套用在所有 h1 header
